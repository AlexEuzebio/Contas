# Contas
Projeto Bootcamp everis-Kotlin

### Descrição:
App Android/Kotlin de cadastro de contas a pagar com os dados de:
- Descrição
- Estabelecimento/Credor
- Valor
- Emissão
- Vencimento
- Pagamento

### Conceitos aplicados:
- HelperDB
- ViewModel
- Thread/UiThread
- DialogFragment
- String Resources
- Styles Resources
- RecyclerView/Adapter/ViewHolder
- Exceptions
- Data Class
